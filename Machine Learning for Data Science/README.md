# Machine Learning for Data Science

## Assignment Summary

This computer lab investigates key machine-learning concepts through simulation exercises, starting with linear regression excess-risk analysis in random design.

## What The Work Does

- Derives and discusses risk definitions for supervised learning.
- Simulates Gaussian design linear models with additive Gaussian noise.
- Estimates OLS repeatedly to empirically validate the theoretical excess-risk scaling:
  $E[R(\hat\theta)] - R(\theta^*) \approx \sigma^2 (p+1)/n$.
- Produces histograms/scatter diagnostics and confidence summaries across Monte Carlo repetitions.
- Continues with additional guided lab questions in the same notebook.

## Main Files

- `MachineLearning_ComputerLab.ipynb`: complete lab notebook with theory, code, and plots.

## Typical Output

- Simulation-based empirical evidence for theoretical generalization-risk behavior.
- Figures and summary statistics for each question in the lab.
