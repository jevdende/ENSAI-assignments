# ENSAI Assignments

This repository contains coursework for the Master in Smart Data Science at ENSAI.
Each top-level folder corresponds to one course assignment (or set of labs).

## Course Index

| Course | Assignment focus | Main artifacts |
|---|---|---|
| [Data Visualization](Data%20Visualization/README.md) | Interactive sports analytics presentation with Bokeh | `data_viz_bokeh_slides.py`, `data/` |
| [Dimension Reduction](Dimension%20Reduction/README.md) | Low-rank matrix completion on movie ratings | `Dimension_reduction_lab.ipynb` |
| [Functional Data Analysis](Functional%20Data%20Analysis/README.md) | Imputation for sparse functional data and method comparison | `compare_methods.R`, `pace.R`, `mice.R`, `missforest.R` |
| [High Dimensional Time Series](High%20Dimensional%20Time%20Series/README.md) | Air quality time-series preprocessing and clustering in Taiwan | `descriptive_analysis.R`, `clustering_analysis.R`, `clustering_analysis2.R` |
| [Machine Learning for Data Science](Machine%20Learning%20for%20Data%20Science/README.md) | Empirical validation of regression risk bounds and simulation exercises | `MachineLearning_ComputerLab.ipynb` |
| [Machine Learning for NLP](Machine%20Learning%20for%20NLP/README.md) | Named Entity Recognition with LSTM and BERT | `projet-2025.ipynb` |
| [Parrallel Computing with Python](Parrallel%20Computing%20with%20Python/README.md) | Multiprocessing and Dask performance labs | `Lab1/`, `Lab2/` |
| [Spark](Spark/README.md) | Large-scale gas-price modeling with PySpark | `Spark_project.ipynb`, `maps/`, `plots/` |

## Notes

- Most assignments are delivered as notebooks or scripts and include exploratory analysis, model development, and visual outputs.
- Some folders contain generated figures or maps produced by the analysis workflows.
- Course-specific details, data expectations, and outputs are documented in each folder README.
